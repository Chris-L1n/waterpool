## 二、会产生哪些 CSV 文件

程序运行后，在项目目录下自动创建 `csv文件/` 文件夹。每次运行生成一个以启动时间命名的子文件夹，里面放 4 个 CSV 文件：

新csv文件不会覆盖上一次csv。 
每次启动自动创建新的 `csv文件/”当前时间“/` 文件夹，旧文件夹自动保留。 
用时间戳区分每次的记录

```
csv文件/
└── “当前时间” 比如（20260626_143025）/          ← 启动时间（年月日_时分秒）
    ├── radar_targets.csv     ← 全部雷达目标
    ├── radar_nearest.csv     ← 最近目标
    ├── radar_tracks.csv      ← 带航迹的全部目标
    └── anomaly_events.csv    ← 异常报警记录
    |__ radar_filtered.csv     过滤后的数据
```
