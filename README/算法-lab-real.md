
**全部恢复。**

|------|--------|------|
| `duration_threshold_s` | **15.0** | 实验方案文档要求 |

## 三、KA 船靠岸送人上岸

### 实验室版（当前代码）

位置：`anomaly_detector.py` `_check_shore_docking()` 第 440-600 行

双路径，任一触发即报警：

**路径A（偏航+减速停留）：**

```
水池太小，不考虑授权点

① was_moving=True（必须曾经移动过 → 排除墙壁固定反射）
② X坐标 偏离航道中心线（normal_channel） >0.2m（偏航）
③ （速度检查已禁用，回到中心线即清零）//速度 <0.2m/s（减速停止） 
④ 持续 ≥ 3 秒 → 报警
```

**路径B（运动特征，不依赖区域多边形）：**

```
水池太小，不考虑授权点

① was_moving=True（曾移动过）
① 距 unauthorized_docking_zone 边界线 <0.5m （进入 Y<1.0m 区域））算到达岸边
② 持续 ≥ 3 秒 → 报警
```

具体路径B：

船在水池里跑
  ↓
每帧记住船的位置 + 距岸距离（保留最近5秒）
  ↓
船离 unauthorized_docking_zone 边界线 < 0.5m？
  ↓ 是
翻看刚才5秒的记录：
  - 平均速度 ≥ 0.3 m/s ？（确认是冲过来的，不是飘过来的）
  - 距离缩短了 ≥ 0.3 m ？（确认确实在靠近岸）
  ↓ 两个都是
标记为"快速接近岸边"
  ↓
船现在速度 < 0.2 m/s ？（停下来了）
  ↓ 是
停了 ≥ 4 秒？
  ↓ 是
💥 报警


---------------------------------------------------------------------------

**可以去掉 `was_moving` 保护**（海面无固定反射）。路径A 逻辑不变，路径B 的坐标系改为 GPS。参数放大到港口尺度：

路径A（偏航+减速停留）
路径B（运动特征，不依赖区域多边形）


| 参数 | 生产值 |
|------|--------|
| `deviation_threshold_m` | **10~50m** |
| `speed_threshold_m_s` | **0.5~2m/s** |
| `duration_threshold_s` | **10~60s** |
| `shore_arrival_distance_m` | **20~50m** |

**实现方式：** 路径A 删除 `was_moving` 判断；JSON 参数放大。路径B 改动较大——需要引入 GPS 和电子围栏 AOI 替代多边形距离计算。

## 四、滤波算法

### 实验室版（当前代码）

滤波分两层，串联执行：

#### 第一层：SimpleTracker（ID 跟踪）— `radar_track_parser_v4.py`

```
输入：雷达每帧输出的原始目标（无稳定ID的目标列表）
输出：带 track_id 的目标列表

算法：最近邻匹配
  - 当前帧每个目标，在上一帧所有活跃轨迹中找距离最近的
  - 距离 < match_threshold_m(0.4m) → 同一目标，继承旧 ID
  - 找不到匹配 → 新目标，分配新 ID
  - 目标消失 < max_missed(20帧=1.0s) → 保留ID，missed+1
  - 目标消失 > max_missed → 删除ID
```

#### 第二层：BoatTargetSelector（船只过滤）— `boat_target_filter.py`

```
输入：SimpleTracker 输出的带 track_id 的目标列表
输出：过滤后的"船只目标"，交给异常检测器 + VOFA+

6 个过滤条件（从配置读取，null=不启用）：

① XY 位置范围
   x_min_cm=-60, x_max_cm=60   → 水池宽度 1.2m
   y_min_cm=0, y_max_cm=400    → 水池长度 4m
   超出范围 → 丢弃（墙壁/天花板反射）

② PV 信号质量 ≥ 30
   弱信号 → 丢弃

③ 匹配距离 < 0.15m
   当前帧位置与上一帧匹配位置的偏差 >0.15m → 丢弃（抖动噪声）

④ 最大速度 < 200cm/s
   雷达偶尔跳到 11m/s，200cm/s 排除极端毛刺

⑤ 最小总位移 ≥ 0.3m（最强过滤）
   目标从首次出现到现在，总位移 <0.3m → 丢弃
   等待 15 帧(1.5s)后才判断，避免误杀慢启动的船
   墙壁反射位移 ~0.03m vs 船 2m+

⑥ 最小存续帧数 ≥ 10 帧
   短命目标 → 丢弃（排除偶然噪声）
```

#### 锁轨机制

```
lock_track=true：锁定后不轻易切换目标
  - 从候选船中按评分函数选最佳目标（hits×100 + PV×0.2 - speed×0.01 - distance×0.1）
  - 锁定目标持续存在 → 始终选它
  - 锁定目标消失 > lock_stale_packets(15帧=0.75s) → 释放锁，重新选择
```

#### 实时运行流程 — `radar_live.py`

```
CAN帧 → 雷达组包 → parse_target_packet() → 原始目标列表
  ↓
SimpleTracker.update() → 分配 track_id
  ↓
BoatTargetSelector.select() → 过滤后选一条船作为 nearest
  ↓
boat_targets = 所有通过过滤器的目标（可能多条船）
  ↓
AnomalyDetector.feed(boat_targets) → SC/KS/KA 检测
  ↓
VOFA+ UDP 转发 boat_targets → 实时点云可视化
```
