# 异常行为检测 — 架构说明

---

## 一、系统总览

```
                      雷达
                       │
                  CAN 帧 (0x421)
                       │
                       ▼
              radar_track_parser_v4.py
              ├─ CAN组包（拼成完整雷达包）
              ├─ 协议解析（提取 X/Y/速度/PV）
              ├─ SimpleTracker（分配 track_id + 过滤随机噪声）
              └─ CSV 写入
                       │
                       ▼  targets（所有检测点，带 track_id）
                       │
                       ▼
              BoatTargetSelector._passes_filters()
              过滤：PV太低？位置在水池外？速度异常？
                       │
                       ▼  boat_targets（只剩可能是船的目标）
                       │
                       ▼
              AnomalyDetector.feed()
              检测 SC/KS/KA 三种异常
                       │
         ┌─────────────┴─────────────┐
         ▼                           ▼
      无异常                       检测到异常
      静默，什么都不做               │
                              ├─ 写 anomaly_events.csv
                              ├─ 终端打印报警
                              ├─ BoatTargetSelector.select() 选出最稳定的目标
                              ├─ on_anomaly() → 触发摄像头转向预置位
                              
```
