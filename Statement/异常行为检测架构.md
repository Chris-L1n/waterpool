## 一、系统总览

```
                      雷达
                       │
                  CAN 帧 (0x421)
                       │
                       ▼
              radar_track_parser_v4.py
              ├─ CAN组包（拼成完整雷达包）
              ├─ 协议解析（提取 X/Y/速度/PV）
              ├─ SimpleTracker（分配 track_id + 过滤随机噪声）
              └─ CSV 写入
                       │
                       ▼  targets（所有检测点，带 track_id）
                       │
                       ▼
              BoatTargetSelector._passes_filters()
              过滤：PV太低？位置在水池外？速度异常？
                       │
                       ▼  boat_targets（只剩可能是船的目标）
                       │
                       ▼
              AnomalyDetector.feed()
              检测 SC/KS/KA 三种异常
                       │
         ┌─────────────┴─────────────┐
         ▼                           ▼
      无异常                       检测到异常
      静默，什么都不做               │
                              ├─ 写 anomaly_events.csv
                              ├─ 终端打印报警
                              ├─ BoatTargetSelector.select() 选出最稳定的目标
                              ├─ on_anomaly() → 触发摄像头转向预置位
                              └─ VOFA+ 实时显示
```

## 二、四个核心算法

### 算法1：CAN组包（RadarReassembler）

文件：`radar_track_parser_v4.py` 第214-266行

把零散的CAN帧拼成完整雷达协议包。每帧8字节，通过分段标记（首段/中段/末段/单段）拼接。

### 算法2：目标跟踪（SimpleTracker）

文件：`radar_track_parser_v4.py` 第355-433行

给每帧的检测点分配跨帧稳定的 `track_id`。雷达本身不给ID，算法用最近邻匹配（连续两帧距离<0.4m即为同一目标）。

**进入异常检测的门槛：** 没有 `track_id` 的点直接跳过，不会进入异常检测。

### 算法3：目标选择（BoatTargetSelector）★ 新增

文件：`boat_target_filter.py`

工作逻辑：

```
每帧所有带 track_id 的目标
    │
    ├─① 记录：更新每个 track_id 的连续检测帧数（track_hits）
    │
    ├─② 过滤：可选条件（位置范围、速度上限、最低PV、最大匹配距离）
    │     默认全部为null，不启用任何过滤
    │
    ├─③ 确认：track_hits ≥ min_track_hits(3帧) 才算"稳定目标"
    │     排除只出现一两帧的噪声
    │
    ├─④ 锁定：如果之前锁定了一个目标，且它仍在候选列表中，继续选它
    │     避免每帧切换目标导致摄像头来回转
    │
    ├─⑤ 评分：track_hits × 100 + PV × 0.2 - 速度 × 0.01 - 距离 × 0.1
    │     选分数最高的（最稳定 + 信号最强）
    │
    └─⑥ 输出：1个目标对象，附加 boat_candidate=True, boat_score, track_hits
```

### 算法4：异常检测（AnomalyDetector）

文件：`anomaly_detector.py`

实时检测三类异常，全海域检测不限定区域。

---

## 三、三类异常行为

| 编号 | 名称 | 判定逻辑 |
|------|------|----------|
| SC | 船-船搭靠 | 接近距离下降 ≥0.5m → 两船距离 <0.5m + 相对速度 <0.3m/s → 持续 ≥15s |
| KS | 快速通过 | 速度 >1.5m/s + 持续 ≥2s +（速度突增 ≥2倍 或 加速度 >0.3m/s²） |
| KA | 船靠岸送人上岸 | 路径A：偏航+减速停留 ≥10s。路径B：快速接近岸边+到达后低速停留 ≥10s |
